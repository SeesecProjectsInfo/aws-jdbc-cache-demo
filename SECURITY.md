# Security Policy

## Supported Versions

Currently, only the `main` branch is actively supported with security updates.

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Instead, please send an email to the repository maintainer or open a private security advisory on GitHub. We will acknowledge receipt of your vulnerability report within 48 hours and strive to resolve it quickly.

If you report a confirmed vulnerability, you may be eligible for public acknowledgement once the patch is released.
